### Week 1 Objectives:
* Start the learning journey, understand the core concepts of Cloud Computing and AWS.
* Practice creating an AWS Free Tier account, setting up basic security (MFA), and configuring cost alerts (AWS Budgets).
* Study and configure advanced access controls using IAM Users, Groups, Policies, and Roles.
* Configure temporary access via Switch Roles, manage session timeouts, and monitor activity with AWS CloudTrail.
* Enforce IAM best practices and clean up temporary practice resources.
